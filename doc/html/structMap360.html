Detailed Description

This class defines a pose-graph map of spherical RGB-D keyframes. This map can be arranged in topological areas defining places with respect to any criteria (e.g. mutual visibility or the description of the image content). This map is structured in a hybrid represetation, as the spherical RGB-D keyframes are annotated with metric, topological and semantic attributes.
