Detailed Description

This class contains the functionality to load the calibration parameters of the omnidirectional RGB-D device (RGBD360). The intrinsic calibration is loaded from previous models obtained with CLAMS (http://cs.stanford.edu/people/teichman/octo/clams/). The extrinsic calibration is loaded from the models obtained with the Calibration programs.
